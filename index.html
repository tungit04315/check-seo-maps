Cần chạy qua Web Server
Browser chặn API calls khi file được mở trực tiếp (file://).
Chạy một trong các lệnh bên dưới trong thư mục chứa file, sau đó mở localhost trong browser.
🐍 Python (có sẵn trên mọi máy)
python -m http.server 8080 click to copy
📦 Node.js / npx
npx serve . click to copy
🔴 VS Code
Cài extension "Live Server" → chuột phải file → Open with Live Server –
Hoặc dùng nút ⚡ Demo để xem thử không cần API.
